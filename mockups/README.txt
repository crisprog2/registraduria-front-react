Descripción: Este archivo es una imagen de referencia para el rediseño de la página principal (HomePage) del proyecto. Utiliza este diseño como guía visual para los estilos, estructura y componentes de la página principal.

Instrucciones para el desarrollador:

- Mantén la lógica de consumo de API y navegación intacta.
- Si tienes dudas sobre algún elemento visual, consulta con el usuario antes de asumir.
